## 五、技术实现声明

1. **算法来源**：本工具使用的图像解密算法由社区与机器学习模型通过密码分析（Cryptanalysis）技术自主推导，分析对象为用户合法导出的游戏截图文件（密文）及用户自有 UID（密钥）。开发者从未对《无限暖暖》游戏客户端实施反向工程、反编译或调试。

2. **算法保护**：为防止恶意使用，与游戏相关的解密算法不开源，仅以编译后的二进制形式提供。此举旨在防止算法被用于非法破解游戏资源、破坏游戏平衡或侵犯知识产权。

3. **无规避行为**：本工具不涉及对任何技术保护措施（TPM）的规避或破解。算法推导基于纯密文攻击（Ciphertext-Only Attack）的数学原理，属于公开的密码学研究范畴。

4. **数据所有权**：工具仅处理用户自行生成的内容（游戏内相机参数），不提取、解密或访问游戏核心资源（模型、贴图、剧情等）。
