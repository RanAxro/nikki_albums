<h2>🌟 Your Reliable Game Album Management Tool</h2>

Nikki Albums is a desktop utility for managing all your in-game albums and share codes from *Infinity Nikki*.

🚀 **Current Version**: 3.010
- **GitHub Download**: [Click here](https://github.com/RanAxro/nikki_albums)
- **Official Website Download**: [Click here](https://nikki.ranaxro.com/en)

💡 **Community & Bug Report Group**: 1062670402
Join our group to share usage tips with fellow players, report encountered issues, and help us continuously improve and iterate this software.

***

# 🎯 Core Features
- **Automatic Game Detection**: No manual configuration required. The program automatically locates your game directory and quickly loads album resources.
- **Multi-Account Isolated Management**: Organize albums for multiple game accounts independently without data interference, keeping your photo library neatly sorted.
- **Full Album Coverage**: Supports management for all 19 regular albums plus 3 in-game resource albums.
- **Batch Processing Tools**: Perform bulk backup, restoration, transfer, deletion, copying and moving for album photos to drastically boost your management efficiency.
- **Duplicate Photo Auto-Cleanup**: Automatically filter redundant duplicate images generated by the game during photo moving or deletion, keeping your album tidy.
- **Basic Image Editing**: Built-in tools for simple image cropping and color grading.
- **Local Network File Transfer**: Transfer images seamlessly across devices within the same local area network.
- **Convert Videos to Live Photos / GIFs**: Transcode videos from the Motion Album or external video files into Android/iOS Live Photos or animated GIF files.
- **Album Metadata Decoding**: View shooting coordinates and camera parameters to easily recreate your perfect in-game shots.
- **Parameter Manager**: Store, parse and manage camera configurations, outfit codes and home space codes.

# Changelog
## v3.010
### New Features
1. Added search function in Parameter Manager; searchable targets include names, tags, clothing names, suit names, lighting presets and filters.
2. Auto strip plain text and irrelevant symbols when pasting or manually inputting share codes.
3. New feature to generate QR codes for outfit sharing codes.
4. Outfit code history records are sorted in reverse chronological order (newest first).
5. Support custom storage path for parameter data files.

### Performance & Optimizations
1. Overall UI optimization.

### Bug Fixes
1. Fixed abnormal color display issues on the dye palette under certain scenarios.
2. Fixed incorrect display of texture names.
3. Fixed wrong rendering of base color parameters for texture drawing.
